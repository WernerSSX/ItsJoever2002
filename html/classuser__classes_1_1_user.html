Detailed Description

An abstract base class representing a generic user in the hospital management system.

The User class contains common attributes and methods shared by all user types, such as hospital ID, password, name, date of birth, and gender. Specific user roles (e.g., Patient, Doctor) should extend this class and implement role-specific behavior.
